# Rock Paper Scissors

A command-line Rock Paper Scissors game where you play multiple rounds against a CPU that picks randomly. Tracks results across the number of games you choose.

## Description

This program asks the user how many games they'd like to play, then runs a loop where the CPU randomly selects rock, paper, or scissors and the user inputs their own choice. After each round, the winner is announced based on standard Rock Paper Scissors rules, and the game continues until the chosen number of games is completed.

## Concepts Covered

- Loops (`while`)
- Conditionals (`if`, `elif`, `else`)
- The `random` module
- User input (`input()`)
- String methods (`.lower()`)
- Input validation
- F-strings

## Goals

- Practice using loops to control program flow
- Get comfortable handling and validating user input
- Use the `random` module to simulate decision-making
- Apply conditional logic to compare multiple variables
- Build a complete, playable mini-game from scratch
