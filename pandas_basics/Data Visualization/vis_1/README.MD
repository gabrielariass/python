# Ice Cream Ratings — Data Visualization

A Jupyter Notebook exploring different chart types in pandas and matplotlib using an ice cream ratings dataset.

## Description

This notebook loads a dataset of daily ice cream ratings and uses it to practice a variety of visualization techniques. It covers seven chart types — line, bar, scatter, histogram, box, area, and pie — giving a broad overview of how to visually represent data depending on what you want to communicate.

## Concepts Covered

- The `pandas` library
- The `matplotlib` library
- The `numpy` library
- Reading CSV files and setting an index
- Line charts (`kind='line'`)
- Stacked bar charts (`kind='bar'`, `stacked=True`)
- Scatter plots (`.plot.scatter()`)
- Histograms (`.plot.hist()`)
- Box plots (`.boxplot()`)
- Area charts (`.plot.area()`)
- Pie charts (`.plot.pie()`)

## Goals

- Get hands-on with the most common chart types in pandas
- Understand when to use each visualization based on the data
- Practice customizing charts with titles, labels, and figure sizes
- Build a foundation for data storytelling and exploratory analysis
